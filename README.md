# springboot-basic

<b>[브랜치]</b>
* only-aply-java 브랜치: 순수한 자바 코드만으로 DI를 적용한 예제
* only-inject-bean-yourself 브랜치: 스프링 빈 등록을 자바 코드의 @Bean이나 XML의 <bean> 등을 통해서 설정 정보에 직접 등록
